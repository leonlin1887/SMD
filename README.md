#  Processing step
1. Delete emails on Application or Webisite
2. Compact OST file  

---

#  Step 1: Delete Emails (Recommended: Web)

##  Method: Outlook Web

 Open:
https://outlook.office.com

###  Filter large emails:
hasattachments:yes
or
size:>10MB


###  Action:
- Select multiple emails
- Click Delete

---



#  Step 2: Compact OST File (Critical)

 Perform in Outlook Desktop

### Path:
File
→ Account Settings
→ Account Settings
→ Data Files
→ Select Account
→ Settings
→ Advanced
→ Outlook Data File Settings
→ Compact Now

---

#  Notes
- Do NOT close Outlook during compact
- Takes 5–30 minutes
- OST size reduces only after this step

---

#  Best Practices
- Keep OST < 30GB
- Use archive mailbox
